# JSpecify 中文文档

JSpecify 是 Java 静态分析的标准注解。本仓库是 JSpecify 官方文档的中文翻译版本。

## 在线访问

[https://deigmata-paideias.github.io/jspecify-zh/](https://deigmata-paideias.github.io/jspecify-zh/)

## 原文地址

- 官方网站: [https://jspecify.dev/](https://jspecify.dev/)
- GitHub 仓库: [https://github.com/jspecify/jspecify](https://github.com/jspecify/jspecify)

## 本地运行

```bash
npm install
npm start
```

此命令启动本地开发服务器并打开浏览器窗口。大多数更改会实时反映，无需重启服务器。

## 构建静态网站

```bash
npm run build
```

此命令将静态内容生成到 `build` 目录，可以使用任何静态内容托管服务进行部署。

## 技术栈

- [Docusaurus](https://docusaurus.io/) - 现代静态网站生成器
